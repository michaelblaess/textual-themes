    Quick start

    pip install git+https://github.com/michaelblaess/textual-themes.git
    from textual_themes import register_all

class MyApp(App):
    def __init__(self):
        super().__init__()
        register_all(self)
        self.theme = "c64"

    Browse all themes interactively in the bundled demo:

    python -m textual_themes
